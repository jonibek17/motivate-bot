# 🌟 Telegram-бот: Мотивация с ИИ и Админ-панелью

Красивый Telegram-бот, который отправляет пользователям вдохновляющие мотивационные цитаты (по 2-4 строки) на выбранном ими языке в случайное время каждый день. Проект включает в себя стильную веб-панель администратора (Glassmorphism Dark Theme) для мониторинга активности.

---

## 🛠️ Технологический стек

- **Backend:** Node.js, Express, Telegraf (Telegram Bot Framework), Node-Cron.
- **AI-генерация:** Google Gemini API (модель `gemini-1.5-flash`).
- **База данных:** Neon (PostgreSQL) с поддержкой автоматического переключения на локальный JSON-файл для тестирования без базы данных (Demo Mode).
- **Frontend (Админ-панель):** React, Vite, чистый CSS (адаптивный дизайн, Glassmorphism, SVG-диаграммы).

## 🚀 Быстрый запуск (Локально)

### 1. Установка зависимостей

Установите зависимости для бэкенда (в корне проекта) и фронтенда:

```bash
# Установка зависимостей бэкенда
npm install

# Установка зависимостей фронтенда
cd frontend
npm install
cd ..
```

### 2. Настройка окружения (`.env`)

Создайте или отредактируйте файл `.env` в корне проекта. Вставьте туда свои токены:

```env
TELEGRAM_BOT_TOKEN=your_bot_token_here
GEMINI_API_KEY=your_gemini_api_key_here
DATABASE_URL=postgres://...  # Оставьте пустым для Demo-режима (будет использоваться локальный JSON-файл)
PORT=5000
```

> 💡 **Как запустить в Demo-режиме (без Neon PostgreSQL и Gemini)?**
> Просто оставьте поля `DATABASE_URL` и `GEMINI_API_KEY` пустыми. Бот будет использовать локальный файл `database/db_local.json` как БД и выбирать цитаты из встроенного русско-английского каталога.

### 3. Запуск в режиме разработки (Development)

Для разработки мы запускаем бэкенд и фронтенд параллельно.

1. **Запустите бэкенд** (в корне проекта):
   ```bash
   npm run start
   # или npm run dev (если установлен nodemon)
   ```

2. **Запустите фронтенд** (в папке `frontend`):
   ```bash
   cd frontend
   npm run dev
   ```
   Фронтенд будет доступен по адресу: [http://localhost:3000](http://localhost:3000). Все запросы к `/api/*` будут проксироваться на бэкенд (порт 5000).

## 📦 Сборка и запуск в продакшене (Production)

Для развертывания проекта на сервере (например, Render, Heroku или VPS) нужно скомпилировать фронтенд. Бэкенд настроен так, чтобы раздавать скомпилированные статические файлы автоматически.

1. **Соберите фронтенд:**
   ```bash
   cd frontend
   npm run build
   cd ..
   ```
   Скомпилированные файлы появятся в `frontend/dist`.

2. **Запустите единый сервер:**
   ```bash
   npm start
   ```
   Теперь Express-сервер на порту `5000` будет одновременно обслуживать и API бота, и веб-интерфейс админ-панели по адресу [http://localhost:5000](http://localhost:5000).

## 🤖 Команды Telegram-бота

- `/start` — Инициализирует пользователя в базе данных, приветствует его и предлагает выбрать язык (Русский или English).
- `/settings` — Открывает интерактивное меню настроек, где пользователь может:
  - Сменить язык.
  - Включить или выключить ежедневные уведомления.
  - Получить мгновенную цитату в чат нажатием на кнопку «Вдохнови меня сейчас ✨».
